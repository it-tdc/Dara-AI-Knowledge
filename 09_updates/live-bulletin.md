# Live Bulletin — Promo & Operational Updates

> **⚠️ DOKUMEN PRIORITAS TERTINGGI (HIGHEST PRIORITY)**
>
> File ini adalah **sumber kebenaran utama (single source of truth)** untuk informasi operasional dan promosi yang berlaku **SAAT INI**.
>
> **Aturan emas:**
> - Jika ada ketidaksesuaian antara file ini dengan file lain (package, pricing, FAQ, SOP, dll), **file inilah yang berlaku**.
> - Dara **wajib** merujuk ke file ini terlebih dahulu sebelum menjawab soal harga, promo, ketersediaan, kebijakan DP, dll.
> - Setiap update di file ini **langsung berlaku** tanpa perlu menunggu review cycle file lain.
> - File lain akan di-update secara berkala (bulanan) untuk menyesuaikan dengan file ini.
> - Dara **jangan** informasikan promo & program berjalan **jika** tanggal priode berlaku pada promo atau tanggal acara sudah lewat.

---

## 📌 Cara Pakai untuk Dara

1. **Selalu cek file ini dulu** saat customer bertanya:
   - Harga / pakak / promo / diskon
   - Syarat DP / cicilan / pelunasan
   - Ketersediaan tanggal / venue / vendor
   - Hadiah / bonus / program referral
   - Perubahan kebijakan mendadak

2. **Jika informasi ada di sini** → gunakan informasi ini (ini yang paling update).
3. **Jika informasi TIDAK ada di sini** → fallback ke file lain (package, pricing, FAQ) dengan catatan "berdasarkan data terakhir update [tanggal], untuk yang paling update Dara cekkan ke tim ya".

4. **Jika ragu/bertentangan** → gunakan info di file ini, lalu catat untuk dikonfirmasi ke tim.
