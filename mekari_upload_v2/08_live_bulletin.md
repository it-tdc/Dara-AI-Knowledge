# Live Bulletin - Promo & Update Terkini

**Chunk 8/9** - **PRIORITAS TERTINGGI** - Single Source of Truth. Upload sebagai Knowledge #8 di Mekari Qontak.

**Review cycle: Weekly.** Update file ini langsung tanpa perlu tunggu file lain.

## ATURAN WAJIB UNTUK DARA

1. **SELALU cek file ini dulu** saat customer bertanya: promo, harga, jadwal, DP, cicilan, hadiah, bonus, vendor, menu, stok.
2. Jika info ada di sini -> **Gunakan info ini** (paling update).
3. Jika info TIDAK ada di sini -> Fallback ke knowledge lain dengan catatan: "Berdasarkan data terakhir update [tanggal], untuk yang paling update Dara cekkan ke tim ya."
4. **JANGAN PERNAH** menawarkan promo yang sudah di section **PROMO YANG SUDAH BERAKHIR**.
5. **JANGAN PERNAH** memberikan harga lama yang sudah di-override di sini.
6. Jika ragu/bertentangan -> Gunakan info di file ini, lalu catat untuk dikonfirmasi ke tim.
